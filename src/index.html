About Quick Slicer {{version}}

Quick slicer is a prototyping tool to initialize the layout of a web page.

Keyboard Shortcuts

  • ESC/Space: Back to Select mode.
  • H: Draw Horizontal Line.
  • V: Draw Vertical Line.
  • O: Create Offset Inside.
  • E/Delete: Switch to Erase mode.
  • A: Toggle Attributes View palette.
  • B: Toggle between Percentige Ratio and Absolute Dimension.
  • F: Switch the Fill Color for blocks.
  • R: Toggle Round Corner mode for blocks.
  • T: Toggle the display of Composite Id tag at top-left of each block.
  • Z: Toggle dimension measurement between from the start and to the end.
